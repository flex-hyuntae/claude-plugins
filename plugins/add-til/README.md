# add-til

오늘 배운 내용(TIL)을 Notion 데이터베이스에 기록하는 플러그인.

## Skills

| Skill | Description |
|-------|-------------|
| `add-til` | Notion TIL DB에 배운 내용 추가 |

## Usage

```
/add-til 모달 닫기 애니메이션을 위해 open과 type 상태를 분리해야 한다
```

또는 대화 중 "오늘 배운거야", "TIL", "지식 기반에 추가해줘" 등으로 호출.

## Requirements

- Notion MCP 서버 연결 필요
- 대상 DB: `claude 지식 기반` (제목 + 날짜)
