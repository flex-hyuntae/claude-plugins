# coding-conventions

코드 생성 시 참고할 코딩 컨벤션 및 아키텍처 패턴 플러그인.

## 포함 내용

- **컴포넌트 폴더 구조**: `index.tsx` + `style.css.ts` 기본 구조 및 확장 패턴
- **DDD 기반 디렉토리 구조**: 도메인/동작별 그룹핑 원칙
- **코드 품질 기준**: code-review 플러그인 리뷰 기준 참조

## 설치

```bash
claude mcp add-from-claude-plugin /path/to/claude-plugins plugins/coding-conventions
```
