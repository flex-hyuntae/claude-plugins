# Accessibility Best Practices

접근성(a11y) 규칙.

## When to Apply

- UI 컴포넌트를 작성할 때
- 인터랙티브 요소를 구현할 때
- 코드 리뷰에서 접근성을 검토할 때

## Quick Reference

### Accessibility (HIGH)

- `a11y-basics` - 시맨틱 HTML, ARIA 속성, 키보드 네비게이션, 스크린 리더 지원

## How to Use

개별 rule 파일에서 상세 설명과 코드 예시를 확인:
```
rules/accessibility/a11y-basics.md
```

각 rule 파일에는 Incorrect/Correct 코드 예시와 설명이 포함되어 있다.
