# Naming Best Practices

네이밍, 문서화, i18n 컨벤션 규칙.

## When to Apply

- 컴포넌트, 함수, 변수명을 지을 때
- i18n 번역 키를 설계할 때
- 주석이나 문서를 작성할 때

## Quick Reference

### Naming & Conventions (MEDIUM)

- `naming-conventions` - 컴포넌트/함수 네이밍, 이벤트 핸들러 패턴, i18n 키 구조, 주석 규칙

## How to Use

개별 rule 파일에서 상세 설명과 코드 예시를 확인:
```
rules/naming/naming-conventions.md
```

각 rule 파일에는 Incorrect/Correct 코드 예시와 설명이 포함되어 있다.
