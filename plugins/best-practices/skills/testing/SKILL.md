# Testing Best Practices

테스트 코드 품질 규칙.

## When to Apply

- 테스트 코드를 작성할 때
- 코드 리뷰에서 테스트 품질을 검토할 때

## Quick Reference

### Testing (MEDIUM)

- `test-quality` - AAA 패턴, 동작 기반 테스트, brittle test 방지

## How to Use

개별 rule 파일에서 상세 설명과 코드 예시를 확인:
```
rules/testing/test-quality.md
```

각 rule 파일에는 Incorrect/Correct 코드 예시와 설명이 포함되어 있다.
